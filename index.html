Rotating Dinner Plan

Two-Week Cycles, On Repeat

Fourteen distinct dinners per cycle — no repeats inside a cycle. The plan advances itself every other Sunday: this week's Cycle B automatically becomes next period's Cycle A, and a fresh Cycle B rolls in.

This Period at a Glance

Eight “anchor” dinners appear every cycle; six rotate in and out, so a new Cycle B never repeats the cycle it follows. Proteins are spread so you’re never eating chicken (or beef) two nights running.

▲▼ shifts a meal between days/weeks · the dropdown swaps or clears a day · ⇄ moves a meal to the other cycle. Changes save & sync automatically.

Price Editor

Set the real price of every item, per store. Each row has a Unit and a Store dropdown — pick a store and the price you enter is remembered for that store, so you can compare a whole cycle at Costco vs Kroger vs Aldi vs Publix. The selected store's price feeds the cycle totals. Staples are priced per unit (one change updates every recipe that uses them). All edits save in this browser.
